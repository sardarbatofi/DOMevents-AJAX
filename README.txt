Swedish:
Uppgift: Quizmaster
Med hjälp av data från ett öppet API ska du bygga upp en webbsida som hämtar frågor, presenterar dem för användaren, tar emot ett svar och kontrollerar det och visar om användaren svarat rätt eller fel.
Du väljer själv hur frågorna ska presenteras (en i taget eller alla på en gång), i vilken form användaren ska svara och hur svarskontrollen ska presenteras.
När användaren har svarat på alla frågor ska resultatet presenteras (10 av 10, 100% eller hur du nu vill visa det.)


English:

Tasks: Quizmaster
Using data from an open API, build a web page that retrieves questions, presents them to the user, receives an answer, verifies it, and shows if the user answered correctly or incorrectly.
You choose how the questions will be presented (one at a time or all at once), what form the user will answer and how the answer check will be presented.
Once the user has answered all questions, the result will be presented (10 out of 10, 100% or how you want to show it now.)
